# Project 1: ML-Based Stock Classification System

## Overview
A sophisticated machine learning system designed to classify stock actions (Buy/Hold/Sell) using comprehensive market analysis and predictive modeling.

## Key Features
- **Multi-class Classification**: Built ML models to classify stock actions into Buy, Hold, or Sell categories
- **Comprehensive Feature Engineering**: Utilized 250+ market features for robust analysis
- **Advanced ML Algorithms**: Implemented Decision Trees, Random Forests, and Gradient Boosting
- **Feature Selection**: Identified top 10 most impactful features through algorithmic analysis
- **Markov Chain Framework**: Developed predictive framework using Markov Chain methodology
- **High Performance**: Achieved 95% Win Rate on Large & Mid Cap Stocks

## Technical Stack
- **Machine Learning**: Decision Trees, Random Forests, Gradient Boosting
- **Predictive Modeling**: Markov Chain based framework
- **Data Processing**: 250+ market indicators and technical features
- **Target Markets**: Large Cap and Mid Cap stocks

## Performance Metrics
- **Win Rate**: 95% on Large & Mid Cap Stocks
- **Classification**: Multi-class (Buy/Hold/Sell)
- **Feature Optimization**: Top 10 most impactful features identified
