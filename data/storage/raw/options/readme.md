# Index Options Data - NIFTY & SENSEX

This directory contains mid-frequency (1-minute timeframe) historical index options data for **NIFTY** and **SENSEX**.

---

## Time Period Covered

- **Start Date:** October 03, 2024  
- **End Date:** February 10, 2026  
- **Total Duration:** ~16 Months

---

## Underlyings

- **NIFTY Index**
- **SENSEX Index**

---

## Timeframe

- **1-Minute Candlestick Data**

Each row represents one 1-minute candle.

## Data Format

Each .csv file contains OHLCV data for a single index:

| Column | Description |
|--------|-------------|
| Date | Datetime of the data point |
| Open | Opening price |
| High | Highest price during the interval |
| Low | Lowest price during the interval |
| Close | Closing price |
| Volume | Volume traded during the interval |
